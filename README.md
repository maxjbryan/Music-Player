# Java Music player For Linux
Java music player using Swing and a few other libarys.
GUI set up as a miniature desktop enviroment using a JDesktopPanel for playlist and file management.
Integration with yt-dlp for downloading playlists and single songs. 
ability to use songs downloaded into playlists to create user made playlists. 
works in wav files.
